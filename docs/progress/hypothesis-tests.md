# hypothesis-tests — 가설 검증

확증적 시연이 아니라 *결과가 미리 안 보이는* 가설을 데이터로 검정. 강한 신호가 나오면 멈추고 보고.
순서: H-동질화 → H-날씨 → H-확산 → H-회피규칙.

## 2026-06-05 H-동질화 심화 (월 해상도 + 분해) — 이중 과정 발견
사용자 지시로 동질화를 깊게. NEIS는 2021+만 제공(뒤로 확장 불가, 4개교로 확인) → 같은 창에서 월 해상도(60점)로 시간 점 확보. `hypothesis_homogenization_deep.py`. balanced panel 2044교(전북 제외 — 전북은 NEIS 중식이 2024부터라 커버리지 변화가 가짜 동질화를 유발; 단 시도당 ≥5 임계로 이미 분산계산엔 빠져 수치 영향은 없었음).
- 순 분산(월별 탈계절): -4.9%, MK z=-2.76(유의하나 월 자기상관에 과대평가), 학교 부트스트랩 p≈0.56(비유의) → **순 동질화는 robust하지 않음**.
- 핵심(속성별 분해): 지역 간 분산 변화가 상반 → 수렴(일식 -58%·가공 -53%·발효 -50%·찌개·빵)과 발산(돼지고기 +112%·중식 +87%·국·밥·떡)이 **동시 진행**. 순 평탄은 상쇄 결과. 중식(마라) 발산은 앞 'temporal: 마라=영남 주도'·'spatial: 영남 매운'과 교차검증.
- MDS 궤적: 지역들이 각자 군집 유지, 보편적 수렴 없음.
- 결론: '동질화 vs 향토 보존'은 양자택일이 아니라 **축마다 방향이 다른 이중 과정**. caveat: 분해 %는 5년 분산비라 노이즈 크고 순결과는 비유의 → 가장 견고한 건 중식 발산. 5년은 문화 드리프트엔 짧으나 더 긴 baseline은 NEIS 불가. 산출: figures/hyp_homog_{monthly,mds,attrs}.png.

## 2026-06-05 H-확산 — 약함(전국 동시, 확산 아님)
질문: 마라가 수도권→지방으로 확산? `hypothesis_diffusion.py`. 시도×연도 마라율 + 서울 거리(EPSG:5179) 상관.
결과: 거리×조기채택(2021-22) r=+0.22 p=0.39(비유의, 부호도 반대), 수도권 조기채택 7.07 < 비수도권 8.35. 마라는 2021→22 전국이 거의 동시 점프(전국 식자재/메뉴 주도). 오히려 대구·부산·울산(영남)이 최고 — 앞선 영남 매운맛 군집과 일치. → 거리 기반 확산 없음. 강한 신호 아님 → H-회피규칙으로. 산출: figures/hyp_diffusion.png.

## 2026-06-05 H-날씨 — 깨끗한 귀무(날씨 무반응)
질문: 계절 말고 그날 기온에 급식이 반응하나? `hypothesis_weather.py`. 기온=Open-Meteo 7개 도시 일평균(무료·키 불요).
설계: day-of-year 기후값(±7일 순환평활) 제거 후 기온 이상치 vs 국물요리·냉면 비중 이상치 상관(평일·끼수≥500, 1,002 학교일), 2주 블록 부트스트랩 CI.
결과: 기온이상치×국물 r=+0.001, ×냉면 r=+0.023 — 둘 다 0 포함·비유의. 대비: 계절 제거 *전* raw 기온×냉면 r=+0.585(강함, 순전히 계절). → **겉보기 0.585는 전부 계절 교란; 캘린더를 빼면 날씨 효과 0.** 급식은 미리 짠 식단표라 당일 날씨 무반응. 강한 신호 아님 → H-확산으로.
교훈: 통제(계절 제거)가 가짜 상관을 드러낸 교과서 사례. 산출: figures/hyp_weather.png.

## 2026-06-05 H-동질화 — 약함 / 기각 못 함
질문: 표준화 시대에 시도 간 식단 차이가 줄어드나(동질화)? `hypothesis_homogenization.py`.
설계: 연도별 시도 속성 프로파일(CLR) 간 평균 쌍거리 = '지역 분산'. balanced panel(5년 모두 연 30끼+ 학교 2,046개)로 표본 구성 변화 통제, 학교 재표집 부트스트랩(400)으로 CI·기울기 p.
결과: 2021→2025 분산 −5.9%(동질화 *방향*)이나 비단조(2024 저점, 2025 반등). 기울기 −0.0375/년, 부트스트랩 p≈0.43, 95% CI가 0 포함 → **유의하지 않음. 강한 신호 아님 → 다음(H-날씨)으로.**
caveat: balanced panel에서 전북 n=2·세종 n=19 등 소표본 시도가 CI를 키움(과소검정). 점추정도 modest라 결론(뚜렷한 동질화 없음)은 유지. 산출: figures/hyp_homogenization.png.
