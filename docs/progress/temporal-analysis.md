# temporal-analysis — 시간 차원 (계절·트렌드·요일)

날짜(`MLSV_YMD`)는 그동안 미사용이었다. 220만 끼 · 1,980개 날짜(2021-01 ~ 2026-06). 다음 본 작업 주제.

## 2026-06-05 유행이 급식에 반영되는가 — 임베딩 확장 + 유행 vs 건강식
`trend_reflection.py`. 키워드를 손 나열 대신 **FastText 임베딩 중심벡터로 어휘 자동 확장**(유행→마라샹궈·마라탕면·두쫀쿠·두바이쫀득초코떡·탕후루·마카롱…, 건강→채식만두·비건탕수육·콩불고기·곡물불고기·저탄소채식의날·두유). 전북 제외.
결과: (1) **어른(영양사)은 '메뉴화 가능한' 유행만 따라감** — 유행식 33→60(1.8배, 마라 3.5→24·마라탕 2→13.5 급증)이나 간식·디저트형(탕후루·두바이·그릭)은 바닥. 급식 제도가 간식 유행은 못 담음. (2) **경쟁구도 유행식 압승** — 유행식 60 vs 건강식 6.2(10배), 건강식 4.5→6.2로 정체(단 '저탄소채식의날' 등 제도 흔적). (3) 지역 — 유행식 경기·세종·영남(대구) 높고 서울 13위(수도권 62.1 vs 비수도권 60.9 ns, 서울거리 r=−0.34 p=0.19 → '수도권=유행 리더' 약함), 건강식 울산·광주·제주↑. 건강×유행 지역 trade-off 약함(r=−0.26 p=0.34; 울산은 둘 다↑). 산출: figures/refl_{fad_lines,compete_nat,maps,compete_scatter}.png.

## 2026-06-05 트렌드 지역 민감도 — 영남(대구) 선두, 서울 둔감 (반직관)
`trend_sensitivity.py`. 유행 바스켓(마라·마라탕·두바이·탕후루·약과·그릭·바질·비건)의 시도별 z-수용지수(2025 수준 + 2021→25 증가). 전북 제외.
결과: 민감 대구 +0.58·경북 +0.39·광주·대전·울산(영남 선두, 대구 마라 33.5/천끼), 둔감 경남 −0.70·서울 −0.44·세종·제주. 수도권 −0.11 < 비수도권 +0.02, 서울 거리 무관(r=0.04, p=0.89) → **"도시=트렌드 리더" 통념 기각**. 대형 유행(마라 CV 0.19)은 전국 균일, 틈새 유행(비건 0.86·탕후루 0.79·두바이 0.45)만 지역적. 앞 spatial '마라=영남 매운맛'과 교차검증. 산출: figures/trend_sens_{map,traj,rank}.png.

## 2026-06-05 A+B 구현 · 결과
`temporal_analysis.py`. 완전한 해 2021~2025(204만 끼, 1,824일)만 사용. 천 끼당 등장률 기반.
- A 캘린더: 극좌표 '달력' + 월×메뉴 히트맵(행 z, 피크월 정렬). 설 떡국1월 · 정월대보름 오곡/부럼/미역국2월 · 삼복 삼계탕/냉면/수박7월 · 추석 송편9월 · 동지 팥죽12월(12월에만). 데이터 주도 발견 — **가장 계절 타는 메뉴는 제철 과일**(자두/수박7월·거봉/포도8월·단감11월·딸기2월·참외5월). 키워드 안 줘도 데이터가 농산물 달력을 스스로 찾음.
- B 트렌드: 로그 slopegraph + 연 기울기·CAGR·Mann-Kendall. 두바이초콜릿 0→×21.5(2024 바이럴) · 마라 ×6.9(MK10,CAGR62%) · 마라탕 ×6.1 · 탕후루 ×4.3(2023) · 약과 ×2.2 · 그릭 ×2.4 상승; 까르보·리조또 하락. 데이터 주도 — 우유·요구르트·쌀밥 하락, 귀리/잡곡밥 상승(정백→통곡 신호).
- caveat: 데이터 주도 트렌드 상위엔 김치 표기 변경(배추김치/포기김치/자율) 노이즈 → 서사는 부분문자열 키워드로 분리. 2026은 1~6월만이라 제외.
- 산출: figures/temporal_{polar,heatmap,slopegraph}.png. 그래프+플레인텍스트로 전달(포스터 X).

## 2026-06-05 방향 결정: A(캘린더) + B(트렌드)
사용자의 ideal 기준(수학·수치·참신·예쁜 viz·단계적 설명)에 가장 부합하는 조합으로 A+B 채택.
- A 급식의 캘린더: 계절·명절이 식단을 지배하는가. 방법 예정 — STL 분해·계절지수·순환통계. viz — 극좌표 '달력', ridgeline, 월×메뉴 히트맵.
- B 식문화 트렌드: 마라의 부상 등 연도 변화. 방법 예정 — 트렌드 검정·CAGR·변화점. viz — slopegraph·streamgraph.
본 구현 대기(아직 "고" 없음).
