## 2026-06-05 유행이 급식에 반영되는가 — 임베딩 키워드(2회 적대검증 교정)
`trend_reflection.py`. [방법] 구체적 트렌드 root + **root별 임베딩 최근접 변형만 보강(blended centroid 금지) + 스테이플 stopword**. 전북 제외.
[교정 이력 — 중요] 초판은 시드 임베딩 **중심벡터** 확장이라 마라(매운)+탕후루(디저트)+요거트(유제품) centroid가 '디저트 일반'으로 표류 → **아이스크림(3.7만 건)·마카롱·도넛**이 유행식 카운트를 지배(가짜 신호). 약과 확장엔 송편·유과(전통 한과)까지 혼입. root 포함 변형만 허용 + 스테이플 제외로 교정.
[Round1 적대검증 교정] 로제(2021 이미 15.6/천끼 정착·flat)가 바스켓을 지배해 성장률을 짓눌렀고, 건강식 '대체'(치환식 오탐 93%)·'두유'(만두유린기 오탐) 혼입 → 로제·흑당·버블티 제외, 대체·두유 제외.
[최종 결과] 유행식 5.6→30.0(**5.4배** — 로제 제거로 진짜 마라 급증 드러남), 건강식 6.1→10.3(1.7배, 2024→25 하락). **유행:건강 = 2.9배**(10배→3.6배→2.9배로 수렴). → 초판 "둘 다 비슷 성장"은 거짓(5.4 vs 1.7). (1) 개별 라인: 마라 급증, 탕후루·두바이 바닥 → **어른은 '메뉴화 가능한' 유행만 반영**(유지). (2) 지역: 대구·부산(영남) 높고 서울 낮음, 수도권 30.7≈비수도권 30.1, 서울거리 r=−0.18 p=0.52 → **수도권 우세 증거 없음**. 건강×유행 지역 r=−0.46 p=0.07(약한 trade-off). 단 유행·건강 모두 2023 이후 정체. 산출: figures/refl_*.png.
교훈: 임베딩 centroid 표류 + 정착메뉴(로제)·오탐(대체) → 가짜 서사. 키워드 검출도 빈도·추세 적대검증 필요.
[Round2 잔여(경미, 견고성에 영향 없음)]: 유행 바스켓의 79%가 마라 → "유행 반영"은 사실상 "마라(메뉴화 가능 유행) 반영"으로 한정. 그릭=83% 그릭요거트(staple, 바스켓 7%·헤드라인 불변). 건강×유행 r=−0.46은 제주 1개 빼면 p=0.27로 붕괴(단일 시도 의존).

## 2026-06-05 트렌드 지역 민감도 — 지방(호남·영남) 민감·서울 둔감 (마라탕 이중계수 교정)
`trend_sensitivity.py`. 유행 바스켓 시도별 z-수용지수(2025 수준 + 2021→25 증가). 전북 제외.
[적대검증 교정] 바스켓에 마라·마라탕 둘 다라 마라탕(마라 부분집합)이 이중계수 → 제거. 그러자 대구 단독 1위 붕괴.
결과: 민감 광주 +0.46·경북 +0.43·충남 +0.35·대구 +0.34, 둔감 경남 −0.80·서울 −0.36. 수도권 −0.14 < 비수도권 +0.03(견고), 서울거리 r=+0.07 p=0.81 → **"도시=트렌드 리더" 통념 기각**(수도권 둔감은 견고). caveat: 시도 순위는 마라 단일 유행에 민감해 불안정 — 마라탕 이중계수 제거 후 1위는 광주, 대구는 4위(부트스트랩 top-1 ~12%, 제거 전엔 대구 1위 53%였음). 견고한 건 광역 방향(수도권 둔감, 부트스트랩 89%). 지역적 유행은 비건·바질·두바이(순열 p<0.01)뿐 — 탕후루(n=89)·약과 CV는 소표본 우연이라 제외. 산출: figures/trend_sens_*.png.

## 2026-06-05 A+B 구현 · 결과
`temporal_analysis.py`. 완전한 해 2021~2025(204만 끼, 1,824일)만 사용. 천 끼당 등장률 기반.
- A 캘린더: 극좌표 '달력' + 월×메뉴 히트맵(행 z, 피크월 정렬). 설 떡국1월 · 정월대보름 오곡/부럼/미역국2월 · 삼복 삼계탕/냉면/수박7월 · 추석 송편9월 · 동지 팥죽12월(12월에만). 데이터 주도 발견 — **가장 계절 타는 메뉴는 제철 과일**(자두/수박7월·거봉/포도8월·단감11월·딸기2월·참외5월). 키워드 안 줘도 데이터가 농산물 달력을 스스로 찾음.
- B 트렌드: 로그 slopegraph + 연 기울기·CAGR·Mann-Kendall. 두바이초콜릿 0→×21.5(2024 바이럴) · 마라 ×6.9(MK10,CAGR62%) · 마라탕 ×6.1 · 탕후루 ×4.3(2023) · 약과 ×2.2 · 그릭 ×2.4 상승; 까르보·리조또 하락. 데이터 주도 — 우유·요구르트·쌀밥 하락, 귀리/잡곡밥 상승(정백→통곡 신호).
- caveat: 데이터 주도 트렌드 상위엔 김치 표기 변경(배추김치/포기김치/자율) 노이즈 → 서사는 부분문자열 키워드로 분리. 2026은 1~6월만이라 제외.
- 산출: figures/temporal_{polar,heatmap,slopegraph}.png. 그래프+플레인텍스트로 전달(포스터 X).
