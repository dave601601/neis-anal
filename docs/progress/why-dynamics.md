# why-dynamics — 왜 그런가(역학) 외부 리서치

발견의 *무엇*은 데이터로 끝났고, *왜(원인)*는 데이터 밖(급식 제도·정책·세대)에 있다.
외부 다중출처(정부 법령·교육청·학술·언론)를 조사해 패턴①②③을 제도 메커니즘으로 설명. 산출: `DYNAMICS.md`.

## 2026-06-05 패턴④ 추가 — 유행식 vs 건강식 경쟁구도(양방향 거울)

사용자 요청: "유행 반영" 주제의 (2) 헬시플레저 vs 유행 경쟁구도의 *다이나믹스*가 빠졌다 → 서브에이전트 2개(동력 top-down / 한계·정체) 병렬 조사. DYNAMICS.md에 §4로 추가, 종합·한계·§0 표 재정렬(③→④).

[건강식=top-down(강)] 채식·비건 급식은 학생 수요가 아니라 제도가 위에서 주입. 광주 2011 전국 최초 '채식의 날'(환경·전통식생활 명분, 2021 '저탄소 식단의 날' 개명)→2019-21 전국 교육청 확산(울산 고기없는월요일·서울 SOS그린급식 월2회·부산·인천·경남)→조례(제주 2022.3 주1회 완전채식+선택권·부산 2021)→탄소중립 시범학교 102(2021)→340교(2023)·전남 전체 페스코 의무화. 압력주체=교육청·환경부·녹색당·소비자기후행동·채식급식시민연대(인권위 진정 "군대도 채식 주는데")·헌법소원(2021 각하). 전부 성인·제도 → 유행식 bottom-up의 정반대.
[지역차(중)] 건강식 부산·제주·호남 높음 = 채식정책 발원·강도 지도와 일치(호남 최강·제주/부산 조례). 한계: 두부·콩은 전통 식자재일 수도(과대추정), 제주 조례 적합식단 ~20%(선언-실행 괴리).
[정체=학생 거부(강)] 둘 다 2023 정체 이유: 채식의 날 잔반 30-40%↑(인천 영양교사, 경기일보·오마이뉴스·팜인사이트 교차), 만족도 33% 불만, 강원 반대 57.9%(약), 전북 "육류 없으면 밥 안 먹음". 영양우려(B12·단백질·성장, 식품영양학 교수·축산단체) 2차 정당화. 의무→채식선택제 후퇴. 반례: "준비 없는 강제는 거부" — 인천 사전교육 후 81% 찬성 반등.
[핵심 통찰 = 양방향 거울] 유행식=학생이 밀어도 제도(영양기준)가 거름(①) ↔ 건강식=제도가 밀어도 학생 입맛이 거름(잔반↑). 성장률 2.9배 격차=bottom-up 입소문 vs top-down 행정보급 속도차. 둘 다 2023 정체=각기 다른 필터에 동시에 걸림. 급식=두 힘의 합의점.
[한계] 2024→25 하락 직접입증 1차자료 약함(정황). 출처 [²³]~[³⁶] 추가(언론·교육청·환경부·학술).

## 2026-06-05 Why/dynamics 리서치 — 3패턴 병렬 조사 → 통합

[방법] WebSearch/WebFetch 기반 병렬 리서치 에이전트 3개(패턴별), 한국어 키워드 우선. 각 주장에 근거 강도(강/중/약/추측) + 출처 URL. 통합축 도출 = **"급식은 학생 문화의 거울이 아니라 제도가 미리 짠 캘린더"**.

[패턴① 제도 필터 — 왜 메뉴화 가능한 유행만] 근거 강.
- 학교급식법 시행규칙 별표3: 식단 작성 시 **단순당류·식품첨가물 과다 사용 금지** + 에너지 ±10%·탄단지 55~70:7~20:15~30·미량영양소 권장량 이상 → 당류 덩어리 간식은 한 끼 구성 불가(마라는 반찬·국으로 메뉴화돼 통과).
- 어린이 식생활안전관리 특별법 제8조: 학교 내 고열량·저영양 식품(과자·**캔디류·초콜릿류**·아이스크림) 판매 제한 — 탕후루·두바이초콜릿이 정확히 이 범주. [한계] 직접 대상은 '판매'이지 '식단 편성 금지'는 아님 → 별표3이 더 직접 근거, 특별법은 정책 기조 방증.
- 조달 경직성: 식재료 월별 입찰·**매월 27일 계약 체결**, 별표2 원물 분리발주 → 다음 달 메뉴가 전달에 고정. **날씨 무반응(r≈0)과 같은 뿌리 = 사전편성 캘린더**.

[패턴② 수도권 역설 — 왜 지방이 민감] 결론: 재정 인센티브 차이(강).
- 핵심: 친환경·로컬푸드 **추가지원 단가** 경북 1,084·전남 803·인천 554 vs **서울·부산 0원** → 우리 민감(경북·충남·광주)/둔감(서울) 시도와 직접 대응. 전남 2026 차액지원 393억, 서울은 친환경 비율 70→50% 하향.
- 보강(중): 수도권 중앙집중 표준화 공급(서울 66%·경기 95% 일괄) vs 지방 급식지원센터 현물(지역산 다양성). 단 둘 다 '공급' 표준화이지 '편성' 강제는 아님.
- 약/추측: 수도권 학생 외식 대체 접근성(정황만), 영양교사 자율성(전국 공통 → 매개경로).
- [미해소 반례] **경남 둔감**은 재정 단일변수로 설명 안 됨. 상관≠인과. 위탁급식 디저트 침투(수도권 비중)는 모집단 다름.

[패턴③ 전통 후퇴 / 마라=영남] A 강, B 추측.
- A 전통 후퇴 = **세대 입맛 서구화 → 잔반 → 기호도 편성** 연쇄. 청소년 채소섭취 16.6→8.3%(반토막, 질병관리청), 패스트푸드 2배, 김치 반토막. 급식 잔반 **채소찬 57.4%·찌개 22.7%·생선 13.5%**(KCI) + 영양교사 식단 1순위='기호도' → 안 먹는 전통식부터 삭감. 조리인력난(채용률 15%)이 손 가는 나물·발효 가속 제거. **전국 동반 후퇴**(β-수렴 ns)와 방향·시기 일치.
- B 마라=영남 = 영남 매운 식문화(대구 10미)로 정합적이나 **지역별 마라 채택·매운맛 선호 직접 비교 실증 없음 → 추측**. [반례] 전라도도 매운데 핫스팟은 영남 → 미측정 요인(외식 인프라·도시화·인구) 교란 가능.

[종합] 세 역학이 ①의 '미리 짠 캘린더'로 수렴: 무엇이 들어오나(필터①)·어디서 활발한가(예산②)·무엇이 사라지나(세대입맛③). 급식은 '제도가 담을 수 있는 것'만 비추는 거울.
